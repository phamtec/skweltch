Limitations
No support for unicode logging
Litteral 0 cannot be used as null pointer in constraints
Template methods cannot be mocked
Private virtual methods cannot be mocked without spelling out the signature
Compilers without support for variadic macros cannot rely solely on MOCK_METHOD
Compilers without support for variadic macros fail on commas in MOCK_BASE_CLASS
